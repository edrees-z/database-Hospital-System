# Hospital Management System

## Project Overview
This repository contains the database design and implementation for a Hospital Management System. The project was developed as part of my coursework in Computer Engineering to simulate real-world medical data management.

## Technical Scope
- **Database Engine:** Oracle SQL
- **Modeling Tool:** SQL Developer
- **Design Approach:** Focused on Database Normalization (3NF) to ensure data integrity and minimize redundancy.

## Key Features
- **ERD Design:** A comprehensive Entity-Relationship Diagram mapping complex interactions between patients, medical staff, departments, and appointments.
- **SQL Implementation:** Structured scripts for table creation (DDL) and efficient data manipulation (DML).
- **Reporting:** Optimized queries to extract essential medical records and administrative reports.

## Security Considerations
As I am passionate about Cybersecurity, I designed this database with data protection in mind, specifically focusing on:
- **Principle of Least Privilege:** Managing access control at the schema level.
- **Data Integrity:** Implementing strong constraints and validations to prevent unauthorized or inconsistent data entry.

## Future Roadmap
I am constantly working on improving my backend development skills. My goal is to evolve this project into a secure, full-stack application as I continue my journey in global security operations.
